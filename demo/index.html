Provide Directory

Please provide your Roblox cache directory, this is usually stored in %temp%\Roblox on Windows or ~/var/app/org.vinegarhq.Sober/cache/sober on Linux.

If you provide an incorrect directory, you can refresh the webpage.

This data is processed locally, and is not uploaded to any servers.

Consider downloading the standalone version of this tool for automatic directory detection!
